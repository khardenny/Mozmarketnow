# 🇲🇿 MozNow — Marketplace de Moçambique

MozNow é um **marketplace moçambicano moderno** desenvolvido em **HTML + CSS + JavaScript puro**, focado em permitir que clientes encontrem produtos e vendedores verificados publiquem anúncios de forma simples e segura.

O sistema foi desenhado para a realidade de Moçambique, com contacto direto via **WhatsApp**, suporte a produtos **físicos e digitais**, e um painel de administração para gerir vendedores, produtos e clientes.

📍 Baseado na Beira — Sofala, Moçambique.

---

## 🚀 Funcionalidades Principais

### 🛍️ Para Clientes
- Explorar produtos disponíveis
- Filtrar produtos por categoria
- Pesquisar produtos por nome, descrição ou localização
- Ordenar produtos por preço (menor/maior) ou mais recentes
- Visualizar detalhes completos do produto
- Contactar via WhatsApp com mensagem automática

### 🏪 Para Vendedores
- Criar conta como vendedor
- Enviar BI para verificação (upload)
- Aguardar aprovação do administrador
- Publicar produtos com:
  - Nome
  - Preço
  - Categoria
  - Tipo (Físico ou Digital)
  - Localização
  - Descrição completa
  - Até 3 fotos do produto
- Painel de vendedor com estatísticas e produtos publicados

### 🛡️ Painel do Administrador
- Login como administrador
- Dashboard com estatísticas do sistema
- Aprovar ou rejeitar pedidos de vendedores
- Gerir vendedores aprovados
- Gerir todos os produtos publicados
- Gerir clientes registados
- Log de actividade recente
